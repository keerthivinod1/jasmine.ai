# Jasmine.ai

Intelligent fashion & outfit recommendation platform. AI-powered styling assistant with a built-in wardrobe catalog — users set preferences and Jasmine recommends complete outfits by occasion, weather, and style.

## Status
🚧 In active development — built in phases. See `BUILD_PLAN.md` for the roadmap.

## Core Features
- Personalized onboarding (style, colors, budget, climate)
- AI Outfit Generator by occasion, weather, and style
- Weather-based recommendations
- Smart color matching
- Pinterest-style inspiration feed
- Swipe to like/dislike outfits for personalization
- AI packing assistant
- Shopping suggestions
- AI fashion chatbot

## Tech Stack
- **Frontend:** Next.js 14 (App Router) + Tailwind CSS + TypeScript
- **Backend:** FastAPI
- **Database:** PostgreSQL
- **AI/ML:** Scikit-learn / LightGBM, Sentence Transformers, LLM assistant
